# 🚀 Tempo – Smart Task Prioritization Web App

> Tempo isn't just a to-do list — it's a **prioritization engine** that tells you exactly what to do next.

---

## 📌 Overview

Tempo is a web-based task management application designed for students and professionals who struggle with managing multiple deadlines.

Unlike traditional to-do apps, Tempo intelligently prioritizes tasks based on urgency and importance, helping users reduce decision fatigue and stay productive.

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 🔁 Forgot Password (Recovery system)
* 🧠 Smart Task Prioritization Engine
* ⚡ "Do This Next" recommendation system
* 🎯 Task scoring based on priority and deadline
* 🎨 Color-coded urgency indicators (Red / Amber / Green)
* 🔔 In-app notifications for upcoming tasks
* 📋 Add, update, and complete tasks
* 📊 Task tracking (Active / Urgent / Completed)

---

## 🧠 How It Works

Tempo uses a simple yet effective formula:

```
Score = Priority Weight × Deadline Urgency
```

* High Priority = 3
* Medium Priority = 2
* Low Priority = 1

Tasks with higher scores are shown at the top under **"Do This Next"**, along with a clear explanation.
